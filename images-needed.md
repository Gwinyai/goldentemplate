# Image Assets Required for VibeGuide

This document outlines all the image assets needed for the VibeGuide SaaS boilerplate application.

## Brand & Logo Assets

### 1. Main Logo
- **File**: `/public/logo.png`
- **Description**: Primary VibeGuide brand logo for header navigation
- **Dimensions**: 200x60px (recommended), scalable SVG preferred
- **Format**: PNG or SVG
- **Usage**: Site header, footer branding

### 2. Favicon Package
- **Files**: 
  - `/public/favicon.ico` (16x16, 32x32, 48x48px)
  - `/public/apple-touch-icon.png` (180x180px)
  - `/public/favicon-16x16.png`
  - `/public/favicon-32x32.png`
- **Description**: Browser tab icons for all devices
- **Format**: ICO and PNG
- **Usage**: Browser tabs, bookmarks, mobile home screen

## Marketing & SEO Assets

### 3. Open Graph Image
- **File**: `/public/og-image.jpg`

### 4. Hero Section Image (Optional)

## Notes

- All images should follow VibeGuide brand colors: Primary (#6366f1), Secondary (#8b5cf6), Accent (#06b6d4)
- Consider using modern, professional photography style
- SVG format preferred for logos and icons for crisp rendering at all sizes
- WebP format can be used for photos to reduce file size
- All images should be optimized for web (compressed but maintaining quality)
- Consider responsive images with multiple sizes for better performance
- AI-generated or stock photos are acceptable for placeholder content
